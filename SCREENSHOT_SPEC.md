# Screenshot.png Placeholder Specification

## Requirements

The screenshot.png file is required by WordPress to display the theme in the Appearance > Themes section.

## Specifications

- **File Name:** screenshot.png
- **Dimensions:** 1200px width x 900px height (4:3 aspect ratio)
- **Format:** PNG
- **File Size:** Recommended under 1MB for optimal loading
- **Content:** A visual representation of the theme's front page design

## Design Guidelines

The screenshot should showcase:

1. **Header Section**
   - Logo placement
   - Navigation menu
   - Search functionality

2. **Hero Section**
   - Featured property image
   - Search form overlay
   - Call-to-action elements

3. **Property Listings**
   - Property cards in grid layout
   - Property images
   - Price and details display

4. **Footer Section**
   - Footer widgets
   - Copyright information

## Color Scheme

Use the theme's primary colors:
- Primary: #2c3e50 (Dark Blue)
- Secondary: #3498db (Blue)
- Accent: #e74c3c (Red)
- Light: #ecf0f1 (Light Gray)
- Background: #ffffff (White)

## Typography

- Headings: Playfair Display (serif)
- Body: Open Sans (sans-serif)

## Placeholder Instructions

**DO NOT generate the actual image file.**

This document serves as a specification for when the screenshot is created. The actual screenshot should be created by:
1. Taking a screenshot of the installed and configured theme
2. Using design software (Photoshop, Figma, etc.) to create a mockup
3. Ensuring the image meets the specifications above

## File Location

Place the final screenshot.png in the theme root directory:
`nextoolify-real-estate/screenshot.png`
