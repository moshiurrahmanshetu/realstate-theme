== Description ==

This is a child theme for Nextoolify Real Estate. Use this theme to make customizations without losing them when the parent theme is updated.

== Installation ==

1. Install the parent theme (Nextoolify Real Estate) first
2. Upload the nextoolify-child folder to /wp-content/themes/
3. Activate the child theme in WordPress Admin > Appearance > Themes
