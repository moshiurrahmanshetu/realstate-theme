## Versioning Policy

- **Major version (X.0.0)**: Significant changes, breaking changes, or major feature additions
- **Minor version (0.X.0)**: New features, enhancements, non-breaking changes
- **Patch version (0.0.X)**: Bug fixes, security updates, minor improvements

---

## Upgrade Instructions

### From 1.0.0 to Future Versions

1. Backup your website before updating
2. If using a child theme, update the parent theme first
3. Test the update on a staging environment if possible
4. Review the changelog for breaking changes
5. Update any custom code that may be affected
6. Clear all caches after updating
